## Frontend

### Installation

To run the mobile app, follow these steps:

1. Navigate to the `MobileApp` directory:

    ```bash
    cd MobileApp
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

### Usage

Run the app:

```bash
npm start
```

## Backend

### Installation

To run the Backend, follow these steps:

1. Navigate to the `Backend` directory:

    ```bash
    cd Backend
    ```

2. Install dependencies:

    ```bash
    node install
    ```

3. Create a `.env` file:
   
   ```bash
   API_KEY="your_key"
   ```

### Usage

Run the backend:

```bash
npm start
```
